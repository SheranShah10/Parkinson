# Parkinson's Disease Benchmarking Platform

This repository provides a publication-ready AI engineering workspace for benchmarking various machine learning models on the Parkinson's Progression Markers Initiative (PPMI) dataset.

## Supported Architectures
- Traditional Machine Learning
- Deep Learning (CNNs, MLPs)
- Graph Neural Networks
- Multimodal Fusion
- Explainable AI (XAI)
- Longitudinal Progression Prediction

## Installation

```bash
# 1. Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Install pre-commit hooks
pre-commit install
```
